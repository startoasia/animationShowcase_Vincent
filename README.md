# myani.css 動畫效果速查表

本 CSS 檔案 (`myani.css`) 集合了專案中常用的動畫效果。可透過此文件快速查找並預覽現有效果，以便在規劃新功能或頁面時參考使用。

## 使用方式

將對應的 CSS class 添加到需要動畫的 HTML 元素上即可。

## 動畫效果列表

| 效果名稱 | CSS Class | 說明 |
| :--- | :--- | :--- |
| **光線效果** | | |
| 按鈕閃光 | `.light` / `.goldenlight` | 產生一道從左到右劃過的光芒，可用於按鈕或區塊。`.goldenlight` 為金色版本。 |
| **動態效果** | | |
| 果凍跳動 | `.ani-bounce` | 模擬物件像果凍一樣上下彈跳的效果。 |
| 縮放 | `.ani-scale` | 物件會規律地放大再縮小。 |
| 微縮放 | `.ani-miniscale` | 物件會規律地微幅放大再縮小。 |
| 水平漂浮 | `.ani-float-horizontal` | 物件會左右來回緩慢移動。 |
| 垂直漂浮 | `.ani-float-vertical` | 物件會上下來回緩慢移動。 |
| 蹺蹺板 | `.ani-seesaw` | 物件會像蹺蹺板一樣左右搖擺。 |
| 鐘擺搖晃 | `.ani-swing` | 模擬鐘擺從頂部中心點左右搖擺的效果。 |
| **透明度與可見性** | | |
| 閃爍 | `.ani-twinkle` | 物件會緩慢地改變透明度，產生柔和的閃爍效果。 |
| 頻閃 | `.ani-strobe` | 物件會快速地改變透明度，產生強烈的頻閃效果。 |
| 向下淡入 | `.ani-fade-in-down` | 物件從上方淡入並移動到定位。 |
| 放大淡入 | `.ani-zoom-in` | 物件從無到有放大並顯示。 |
| **旋轉與翻轉** | | |
| 旋轉 | `.ani-rotate` / `.ani-rotate-corner` | 物件會持續旋轉。`.ani-rotate` 以中心點旋轉，`.ani-rotate-corner` 以左上角旋轉。 |
| 水平翻轉 | `.ani-flip` | 物件會沿著 Y 軸進行 3D 翻轉。 |
| 垂直翻轉 | `.ani-flip-vertical` | 物件會沿著 X 軸進行 3D 翻轉。 |
| 公轉 | `.ani-revolve` | 物件會像行星一樣繞著一個中心點公轉。 |
| **特殊效果** | | |
| 開箱效果 | `.ani-cover-open` | 模擬兩扇門向外打開，顯示底層內容。 |
| 前後對比圖 | `.BeAf_container` | 透過滑動分隔線來比較兩張圖片的差異。 |
